(접기/펼치기) 입력규칙 적용 / 입력 프로그램 배포 방법

입력규칙 적용방법
1. 입력규칙 생성 버튼 클릭하여 rule.js 파일을 다운로드 받는다
2. 다운로드 받은 파일을 writer/js 폴더 안으로 복사한다

주의: 기존 writer/js/rule.js 파일은 삭제 혹은 다운로드 파일로 교체한다

입력 프로그램 배포 후 실행방법
1. writer 폴더를 압축하여 작업자에게 보내준다.
2. 작업자는 압축해제 후 writer 폴더 안에 research_writer.html 을 실행
3. 구글 크롬 브라우저, IE Edge 브라우저에서 동작 가능
4. 작업자는 설문결과 입력 후 엑셀로 다운로드 하여 해당 엑셀만 취합하여 작업 진행
5. 작업자는 엑셀이 설치되어 있지 않는 경우 다운로드 받은 엑셀을 확인할 수 없음
6. 설문결과 입력 프로그램에서 다운로드 받은 엑셀을 다시 업로드하여 이어서 작업 가능
7. 작업자가 다운로드 받은 엑셀을 엑셀 프로그램에서 직접 수정하는 것은 권장하지 않음
   프로그램에서 다운로드 받은 엑셀이 아닌 다른 엑셀을 업로드할 경우 정상동작하지 않음

주의: Window 7이나 그 이하 버전은 반드시 크롬 브라우저 다운로드 후 실행
(실행할 수 없는 브라우저는 자동으로 크롬 다운로드 사이트로 이동됨)
